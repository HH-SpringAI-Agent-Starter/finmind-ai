# FinMind AI — 需求文档

## 一、产品定位

FinMind AI 是一个面向投研、风控和企业金融数据团队的 **Spring AI 金融分析 Agent**，支持行情查询、财报解读、异动监控、合规输出与可引用财务知识库。

## 二、核心功能需求

### FR-01 实时行情查询
- 支持 A 股、港股、美股股票/ETF/指数实时行情
- Agent 自然语言理解："茅台今天多少钱" → 自动识别为 600519
- 输出包含：最新价、涨跌幅、最高/最低、成交量、换手率

### FR-02 财报解读
- 根据财报文本/PDF 提取关键财务指标
- 支持同比、环比、行业对比分析
- 输出摘要 + 亮点 + 风险点

### FR-03 异动监控
- 实时扫描市场异动（涨幅 > 5%、量比 > 3、换手率异常）
- 用户订阅关注股票，异动时主动推送
- 异动原因推理（板块联动 / 事件驱动 / 资金面）

### FR-04 可引用知识库（RAG）
- 知识来源：财报 PDF、公告、研报、行业分析
- 引用格式：指明来源、段落、数据日期
- 支持向量检索 + 语义搜索

### FR-05 合规输出
- 所有输出必须注明＂不构成投资建议＂
- 区分事实、估算和观点
- 预测性内容附带置信度评分

## 三、非功能需求

### NFR-01 性能
- 对话响应 < 5s（含 LLM 推理）
- 行情查询 < 2s
- 支持 100+ 并发请求

### NFR-02 可扩展
- Agent Tool 插件化注册
- 数据源可配置（替换 akshare / 东方财富 / Wind 等）

### NFR-03 安全
- API Key 通过环境变量注入
- 请求限流（令牌桶）
- 输出内容脱敏（不上传未授权的金融数据）

## 五、数据流

\`\`\`
用户提问 ──→ Agent 意图识别 ──→ 匹配 Tool
    │                        │
    │                   ┌────┴────┐
    │              ┌───┤行情 Tool ├───→ akshare API
    │              │   └─────────┘
    │              │   ┌─────────┐
    ├──────────────┼──→├──财报 Tool├───→ PDF / 数据库
    │              │   └─────────┘
    │              │   ┌─────────┐
    └──────────────┼──→├──异动 Tool├───→ 实时扫描
                   │   └─────────┘
                   │   ┌──────────────┐
                   └──→├RAG 检索 Tool├───→ Vector DB
                       └──────────────┘
                              ↓
                    收集所有结果 → LLM 合成回答
                     → 带引用格式输出 → 返回用户
\`\`\`
